Tecnologias

Tecnologias Front-End

HTML

Hyper Text Markup Language
HTML é a linguagem de marcação usada para criar páginas da web. Ele usa tags e elementos para estruturar e apresentar conteúdo em uma página da web, como texto, imagens, links e outros tipos de mídia. A linguagem HTML é interpretada pelos navegadores da web para exibir o conteúdo da página aos usuários finais. Em resumo, HTML é a base para a criação de páginas da web.

CSS

Cascading Style Sheets
Utilizada para definir a aparência e o layout de elementos em todas as páginas web.

JAVASCRIPT

Usada principalmente para tornar as páginas da web interativas e dinâmicas.

Boostrap

Bootstrap é a framework HTML, CSS e JS mais popular para o desenvolvimento responsivo de projetos mobile first na web. Utilizado por: Twitter, SmartAnalytics, DataMiner, Udacity, Appernetic.io, entre outros.

AngujarJS

AngularJS permite expandir o vocabulário HTML para sua aplicação. O ambiente resultante é extraordinariamente expressivo, legível e rápido para desenvolver. Utilizado por: Wix, The Guardian, SmartAnalytics, DataMiner, Udacity, entre outros.

Tecnologias Back-End

Python

Python é uma linguagem de programação de alto nível, interpretada e orientada a objetos. É conhecida pela sua sintaxe clara e concisa, tornando-a uma das linguagens mais populares entre programadores de todas as habilidades. Python é usada em muitas áreas, incluindo desenvolvimento web, análise de dados, inteligência artificial e automação de tarefas.

Java

Java é uma linguagem de programação popular que é usada em uma ampla variedade de aplicativos, desde aplicativos de desktop até aplicativos da web e dispositivos móveis. Criada pela Sun Microsystems em 1995, Java é conhecido por ser seguro, confiável e portátil, o que significa que o código Java pode ser executado em diferentes plataformas sem a necessidade de alterações significativas.

Ruby

Ruby é uma linguagem de programação dinâmica, interpretada e orientada a objetos que foi criada no Japão em 1995. É uma linguagem de programação popular devido à sua sintaxe simples e elegante, que permite que os desenvolvedores escrevam código de maneira mais rápida e eficiente. Ruby é frequentemente usado para criar aplicativos web.
