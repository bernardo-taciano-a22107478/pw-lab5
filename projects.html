Projetos Universidade

  • Projeto MineSweeper

    O objetivo do projeto era ganhar os primeiros conhecimentos em programação e começar a trabalhar com código. Desenvolvemos o jogo em Kotlin, uma linguagem moderna e poderosa que nos permitiu criar uma aplicação funcional e eficiente. O jogo consistia em encontrar um tesouro escondido em um mapa, seguindo pistas e evitando armadilhas. Além disso, incluímos algumas funcionalidades adicionais, tais como um sistema de pontuação e um modo multiplayer. Durante o processo de desenvolvimento, enfrentamos alguns desafios, mas conseguimos superá-los com a ajuda dos nossos colegas e dos professores. Aprendemos muito sobre programação, trabalho em equipe e resolução de problemas, o que foi fundamental para o nosso crescimento como programadores. Este projeto foi um grande passo no nosso caminho para nos tornarmos programadores profissionais e nos inspirou a continuar a explorar o maravilhoso mundo da programação.

    Ano de realização: 2022
    Cadeira: Fundamentos De Programação
    Participantes: Eu, Rodrigo Taciano
    Tecnologias usadas: Kotlin

  • Projeto LP2

    O projeto desenvolvido em Java para a cadeira de Linguagens de Programação 2 foi uma implementação de um tabuleiro com interface de um jogo de tabuleiro chamado "Deisi Jungle".
    O objetivo era trabalhar com programação orientada a objetos, utilizando conceitos como herança, polimorfismo e encapsulamento.
    O jogo foi implementado com o uso de classes como Board, Player, além das classes específicas para cada tipo de animal, como Leão, Tigre, Elefante, etc. Cada peça tinha suas características e movimentos específicos, e a interação entre elas foi implementada de forma a seguir as regras do jogo.
    Além da implementação das classes e métodos necessários para o jogo, também foi desenvolvida uma interface gráfica para que o usuário pudesse interagir com o jogo de forma visual e intuitiva.

    Ano de realização: 2023
    Cadeira: Linguagens de Programação II
    Participantes: Bernardo Taciano e Rodrigo Pinto
    Repositório no GitHub Vídeo no YouTube
    Tecnologias usadas: Java

  • Projeto Algoritmia e Estrutura de Dados

    O projeto que desenvolvemos para a cadeira de Projeto Algoritmia e Estrutura de Dados consistiu em criar um sistema capaz de processar grandes quantidades de dados em um tempo mínimo, utilizando estruturas de dados como HashMaps, Linked Lists e HashSets.
    O objetivo era criar uma aplicação capaz de receber um grande conjunto de dados através de ficheiros .txt e, em seguida, permitir a realização de consultas e pesquisas nesses dados de forma rápida e eficiente. Para isso, utilizamos estruturas de dados otimizadas para consultas rápidas, como HashMaps e HashSets, que nos permitiram obter resultados precisos e em um tempo muito reduzido.
    O resultado foi um sistema altamente eficiente e preciso, capaz de processar grandes quantidades de dados em um tempo mínimo e fornecer resultados precisos e úteis para os usuários. O projeto permitiu, melhorar as nossas habilidades em programação e desenvolvimento de algoritmos eficientes, além de explorar diferentes técnicas e estruturas de dados para otimizar o desempenho de aplicações em larga escala.

    Ano de realização: 2023
    Cadeira: Linguagens de Programação II
    Participantes: Bernardo Taciano e Rodrigo Taciano
    Vídeo no YouTube
    Tecnologias usadas: Java
